EUDR Polygon Validator

A Python pipeline designed to validate plantation polygon datasets for compliance with the EU Deforestation Regulation (EUDR). Built from real field data collected by MOI FOODS and DASI.

What it checks

  • Geometry validity – self‑intersections, duplicate vertices
  • CRS consistency – ensures all polygons use WGS 84 / UTM
  • Surface area – flags parcels below minimum threshold
  • Overlap detection – identifies duplicate or overlapping claims

Output

Generates a clean, EUDR‑ready dataset with an automated validation report (CSV) highlighting problematic polygons for correction.
